Exporting Results and Reports

Published: January 07, 2026

Once analysis is complete, sharing or preserving results is essential. The Client-Side Readability Checker offers multiple export formats, all generated entirely in your browser with full privacy.

Four options cover common needs: annotated HTML, PDF documents, PNG images, and structured JSON data. Each preserves different aspects of the analysis.

Exports happen instantly—no servers involved. Your text and highlights remain on your device throughout the process.

Available Export Formats

  • HTML: Fully styled with color-coded highlights preserved
  • PDF: Professional layout suitable for printing or sharing
  • PNG: Visual snapshot of the current view
  • JSON: Structured data for developers or further processing

Annotated HTML Export

The HTML option creates a complete webpage with your original text and all interactive-style highlights intact. Colors, tooltips, and formatting match the live analyzer exactly.

Use this for archiving drafts with feedback or sharing detailed reviews with collaborators. The file opens in any browser without dependencies.

PDF for Professional Sharing

PDF exports produce clean, printable documents. Highlights appear as colored backgrounds or underlines, with the summary panel included as a header or footer.

Ideal for formal reports, editing passes, or submitting revised work with evidence of improvements.

PNG for Quick Visuals

The PNG capture saves the current screen as an image. Perfect for quick social media posts, presentations, or embedding in notes.

Scroll position and zoom level affect the capture, allowing focus on specific sections.

JSON for Developers

JSON export provides structured output: full text, sentence array, readability scores, and detailed issue lists. Developers can integrate results into workflows or build custom visualizations.

All exports include timestamps and basic metadata. File names suggest content for easy organization.

Privacy remains absolute—nothing leaves your browser. Even temporary processing happens locally.

Combine exports for comprehensive records: HTML for review, PDF for archive, JSON for automation.

Exporting closes the loop on analysis. Turn insights into actionable, shareable artifacts while maintaining complete control over your writing.
